This project, Cpp-KitLearn, aims to implement popular machine learning algorithms, data manipulation techniques, and deep learning frameworks in C++.

NumPy

Library for numerical operations in Python.

Applications: Array operations, linear algebra, random numbers.

Features: Ndarray, broadcasting, vectorization, and more...

Pandas

Data manipulation and analysis library.

Applications: Data cleaning, manipulation, analysis.

Features: DataFrame, Series, groupby, and more...

Error Calculation

Methods to evaluate the performance of models.

Applications: Model validation, performance metrics.

Metrics: MSE, RMSE, MAE, R2 score, and more...

Classification

Identifying which category an object belongs to.

Applications: Spam detection, image recognition.

Algorithms: Gradient boosting, nearest neighbors,SVM, logistic regression, and more...

Regression

Predicting a continuous-valued attribute associated with an object.

Applications: Drug response, stock prices.

Algorithms: linear Regression and more...

Clustering

Automatic grouping of similar objects into sets.

Applications: Customer segmentation, grouping experiment outcomes.

Algorithms: k-Means, HDBSCAN, hierarchical clustering, and more...

Deep Learning

Advanced machine learning techniques using neural networks.

Applications: Image processing, natural language processing.

Frameworks: TensorFlow, PyTorch, Keras, and more...
